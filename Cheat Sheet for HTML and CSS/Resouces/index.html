HTML and CSS tips and tricks

Cheat Sheet

Tag Name Meaning
<table> Table Tag Defines a table in HTML
<tr> Table Row Tag Defines a row in a table
<th> Table Header Tag Defines a header cell in a table
<td> Table Data Tag Defines a standard cell in a table
<thead> Table Head Tag Defines a group of header rows in a table
<tbody> Table Body Tag Defines a group of body rows in a table
<p> Paragraph Tag Defines a paragraph in HTML
<div> Division Tag Defines a division or section in an HTML document
<span> Span Tag Defines a section in a document
<ul> Unordered List Tag Defines an unordered list in HTML
<li> List Item Tag Defines an item in a list also creates bullet points

Table Attributes

Attribute Meaning
border="value" Specifies the width of the border around the table
cellpadding="value" Specifies the space between the cell content and the cell border
cellspacing="value" Specifies the space between cells in a table
width="value" Specifies the width of the table
height="value" Specifies the height of the table
colspan Specifies the number of columns a cell should span in a table
rowspan Specifies the number of rows a cell should span in a table

HTML Tips

  • Use semantic HTML tags (e.g., <header>, <footer>, <article>) for better structure and accessibility.
  • Always close your HTML tags to avoid rendering issues.
  • Use alt attributes for images to improve accessibility and SEO.
  • Use comments (<!-- Comment -->) to explain sections of your HTML code.
  • Validate your HTML code using online validators to ensure it follows best practices.
  • Keep your HTML code organized and properly indented for better readability.

    • CSS Tips

    • Use external stylesheets for better organization and maintainability.
    • Use classes and IDs to target specific elements for styling.
    • Use shorthand properties to reduce code length (e.g., margin, padding).
    • Use comments to explain complex styles or sections of your CSS.
    • Test your styles across different browsers to ensure compatibility.
